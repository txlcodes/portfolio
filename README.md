# Talha Nawaz - Newspaper Portfolio

A distinctive, newspaper-themed portfolio website for Full Stack Engineer Talha Nawaz. This project features a responsive grid layout, typography-focused design (using Playfair Display, JetBrains Mono, and Inter), and a unique aesthetic reminiscent of a classic print edition.

## Tech Stack

- **React**: UI Library
- **Tailwind CSS**: Utility-first styling
- **TypeScript**: Type safety

## Features

- Newspaper layout with "Print Edition" aesthetics.
- Responsive design (Mobile & Desktop).
- Dynamic data rendering for Experience and Projects.
- Custom "Stamp" and paper texture effects.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm start
   ```
